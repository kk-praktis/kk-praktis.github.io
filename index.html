Übungen

HTML, CSS und JavaScript

    1.
  • Erstelle eine 3x3 HTML-Tabelle.
  • Füge ein Bild in eine Tabellenzelle ein und verlinke es mit einer URL, die sich in einem neuen Tab/Fenster öffnet.
    2.
  • Erstelle ein Formular mit Feldern für Name und E-Mail sowie einem "Senden"-Button. Überprüfe bei der Absendung die E-Mail-Adresse mit reinem JavaScript (Vanilla JavaScript) und zeige eine Benachrichtigung an. Verwende CSS, um ungültige Formularfelder hervorzuheben (z. B. roter Rahmen).
  • Setze die gleichen Funktionen mit jQuery um.
    3.
  • Entwirf einen Taschenrechner mit Schaltflächen für Zahlen von 0 bis 9, Rechenoperationen, Löschen usw. (Tipp: MathJS könnte dabei hilfreich sein.)
    4.
  • Speichere Daten in Cookies und lies sie aus. Übertrage Daten zwischen zwei Seiten und zeige die Daten an.
    5.
  • Erforsche Bootstrap und Tailwind CSS. Was machen sie? Vergleiche ihre Unterschiede und Anwendungsfälle.
  • Wähle eines aus und verwende es in einer der vorherigen oder folgenden Übungen.
    6.
  • Recherchiere zu APIs. Wozu dienen sie? Wie interagierst du mit ihnen? Erkläre REST.
  • Verwende entweder die RAWG- oder TheMovieDb-API
    Du kannst folgende API-Keys nutzen:
    TMDB RAWG
    FRAG-UNS FRAG-UNS
    Lies die Dokumentation und verstehe, wie du die API verwenden kannst.
  • Erstelle eine Webanwendung, die eine der beiden APIs verwendet, um Daten anzuzeigen.
    7.
  • Suche dir einen Datensatz (von Kaggle, Statista etc.) und visualisiere ihn mit einer Diagramm-Bibliothek für JavaScript.

Übungen mit Backend

    1.
  • Erstelle ein Eingabefeld für Ajax-basierte Autocomplete-Vorschläge (nutze eventuell dummyjson.com für Beispieldaten). Erstelle eine Log-Datei und protokolliere jede Suchanfrage darin.
    2.
  • Baue eine TODO-App (Aufgaben erstellen, lesen, aktualisieren, löschen) mit PHP und einer Datenbank.
    3.
  • Implementiere Tages-, Monats- und Jahresansichten in einer Kalenderanwendung. Füge Navigationsschaltflächen hinzu und zeige den aktuellen Monat und das Jahr auf Deutsch an. Stelle alle Tage eines Monats mit abgekürzten Wochentagsnamen dar.
  • Erstelle einen Veranstaltungskalender (Erweiterung der Kalenderanwendung).
    • CMS-Bereich: Implementiere es in einer Klassenstruktur. Verwalte Veranstaltungsarten (z.B. Konzert, Sport) mit Optionen wie Farbe (eventuell mit einem Color-Picker). Erstelle und bearbeite Events (weise Veranstaltungsarten, Titel, Bild, Zeitraum, max. Teilnehmer zu). Verwalte Event-Anmeldungen.
    • Frontend: Zeige Veranstaltungen in der Monatsansicht an (nur Titel) mit einer Filteroption für Veranstaltungsarten. Stelle Detailansichten für Kalendertage dar, die Veranstaltungen auflisten (Titel, Uhrzeit, freie Plätze). Detailansicht von Events beim Anklicken (Bild, Titel, Zeitraum, freie Plätze) mit einem Teilnahmeformular.
  • Erstelle dynamisch einen Feed mit monatlichen Veranstaltungen.
  • Ermögliche es Benutzern, den Feed für jeden Monat zu abonnieren/herunterzuladen.
    4.
  • Erstelle ein Formular zur Newsletter-Anmeldung mit Pflichtfeldern (Name, E-Mail, Zustimmung zu den AGB). Überprüfe die eingegebenen Daten und behandle Fehler. Speichere gültige Daten in einer Datenbank.
    • Administrationsbereich: Füge ein Formular zur Auswahl eines Datumsbereichs hinzu (mit einem JavaScript Date-Picker). Zeige Anmeldungen basierend auf dem ausgewählten Datumsbereich an.
    • Ermögliche das Bearbeiten oder Löschen von Anmeldungen.
    • Exportiere Daten als PDF- oder Excel-Datei.
  • Sichere die gesamte Anwendung mit einem Login-System ab.
    5.
  • Erstelle eine einfache API in PHP (verwende beliebige Daten, z.B. von Kaggle.com oder Statista).
  • Sichere bestimmte API-Endpunkte mit einem API-Schlüssel ab.
    6.
  • Erstelle einen einfachen Twitter-Klon.
  • Benutzer sollten sich registrieren, anmelden, Tweets posten und sie ansehen können.
  • Verwende eine beliebige Backend-Sprache oder ein Framework und beliebige Frontend-Technologien.
